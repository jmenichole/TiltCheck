About This Demo

This demo showcases the vanilla JavaScript version of TiltCheck's overlay system. Unlike the React version, this implementation works with any website without framework dependencies. The overlay provides:

  • Popup Alerts: Top-right notifications for immediate attention
  • AOL-Style Messenger: Bottom-right chat window for ongoing monitoring
  • Stats Dashboard: Top-left real-time statistics display
  • Browser Notifications: Native browser notification support
