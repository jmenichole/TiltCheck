üöÄ Quick Setup

One-Command Setup:

git clone https://github.com/jmenichole/TiltCheck.git
cd TiltCheck
npm install
npm run setup-solana  # Generate Solana wallet
npm run validate      # Check configuration
npm start             # Start integrated mode

Key Environment Variables:

# Required for all modes
DISCORD_BOT_TOKEN=your_discord_bot_token

# For JustTheTip (Solana tipping)
SOLANA_RPC_URL=https://api.mainnet-beta.solana.com
SOLANA_PRIVATE_KEY=your_solana_private_key

# For TiltCheck (Base L2 accountability)
BASE_RPC_URL=https://sepolia.base.org
BASE_PRIVATE_KEY=your_base_private_key
