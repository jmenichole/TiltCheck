## üöÄ Quick Start

```bash
# Clone the repository
git clone https://github.com/jmenichole/TiltCheck.git

# Navigate to directory
cd TiltCheck

# Install dependencies
npm install

# Set up environment
cp .env.example .env
# Edit .env with your Discord bot token

# Start the bot
npm start
```
